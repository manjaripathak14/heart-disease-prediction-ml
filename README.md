# heart-disease-prediction-ml
## Project Overview
This project predicts whether a patient has heart disease using a Random Forest Classifier.

## Dataset
- 1190 records
- 11 features
- Target:
  - 0 = Not Diseased
  - 1 = Diseased

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Model
Random Forest Classifier

## Results
- Accuracy: 95.38%
- ROC-AUC Score: 97.07%
- Recall: 96%
